## :book: Resumo do projeto
Todo API é uma REST API em que o usuário pode gerenciar suas tarefas.

A aplicação possui endpoints para gerenciar e manipular os recursos Todo (Tarefa), User (Usuário), que são protegidos 
e requerem autenticaçao do tipo basic auth para serem manipulados.
